python introduction

“Python库”包含几种不同种类的组件。

“Python库”包含有数据类型，数据类型通常被视为python“核心”的一部分，如数字和列表。对于这些类型， python 语言核心定义了字面形式并对其语义设置了一些限制，但并没有完全定义语义 (另一方面, 语言核心确定了像拼写及操作符优先级这样的语法属性)

“Python库”也包含了内建的 函数 和 异常（这些内建的函数和异常不需要导入就可以被所有的python代码使用 ）。它们中的一些被核心语言定义,但是这里也描述了有很多对核心语义来说不是必须的东西

然而，python库的大部分由模块的集合组成。有许多方法去剖解这些集合. 一些模块是用C写的并且内建在python解释器中; 另外一些模块使用python编写并且以源码的方式被导入. 一些模块提供了高度专注于Python的接口，例如打印堆栈跟踪; 一些模块提供特定于特定操作系统的接口，如访问特定的硬件; 其他一些模块提供特定于特定应用程序领域的接口，像万维网. 一些模块在所有的python版本及ports可用; 其他一些模块仅在底层系统支持或需要它们时可用; 还有一些其他的模块仅当在编译和安装Python时选择特定配置选项时才可用。

本手册“由内而外”组织： 它首先描述了内置的数据类型，然后内置函数和异常， 最后相关联的模块被分组在各个章节之中. 章节的顺序以及每章中模块的顺序大致从最相关到​​最不重要。

这意味着如果您从一开始就开始阅读本手册，并且在无聊的时候跳到下一章，您将得到Python库所支持的可用模块和应用程序领域的合理概述。当然，你不必像小说一样阅读它 — 您还可以浏览目录 …
